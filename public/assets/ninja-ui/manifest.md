# Ninja UI Asset Manifest

This folder contains only the image assets currently used by the ninja typing
game UI. All interface labels, numbers, words, and metrics are rendered in React.

## Usage Rules

- Keep text out of image assets.
- Use the dark icon variants for inactive tabs.
- Use the orange glow variants for active tabs or highlighted decorative marks.
- Preserve the current layered composition unless intentionally redesigning the
  screen.
- If a new asset is added, reference it from code and document it here.
